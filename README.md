### 安装

- git clone https://gitee.com/wangcong2/geekbang-lessons.git
- cd  geekbang-lessons\training-camp\xiaomage-java\stage-0\user-platform
- mvn clean install -U
- cd user-web
- mvn tomcat7:run
- 注意，一定要通过上面的命令启动，否则项目运行不了


### 打开浏览器

- localhost:8080
